h1. 形態素解析エンジンMecab on iPhone（Static Library版）

iPhoneにStatic Libraryとして追加したMecabを利用して、文字列の形態素解析を行うサンプルアプリケーションです。

!http://s3.flcl.jp/libmecab_screenshot_1.png!

h2. ビルド方法

# mecab/mecab.xcodeprojを開きます。
# ターゲット（Device/Simulator）、構成を選びビルドします。
# LibMecabSample/LibMecabSample.xcodeprojを開きます。
# 先ほどビルドしたものと同じターゲット、構成でビルドします。
